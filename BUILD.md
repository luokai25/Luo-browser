# Luo Desktop - Build from Source

## Requirements
- Node.js (v12 or higher)
- Windows (for .exe build)

## Build Steps

1. **Install dependencies:**
```cmd
npm install
```

2. **Test locally:**
```cmd
npm start
```

3. **Build installer:**
```cmd
npm run build
```

The .exe will be in `dist/Luo-Desktop-1.0.0.exe`

---

## Running without Node.js

If you just want to run the app:

1. Download the latest release from: https://github.com/luokai25/Luo-browser/releases
2. Run `Luo-Desktop.exe`
