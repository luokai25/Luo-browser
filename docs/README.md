# Luo Desktop - Live Demo

🌐 **Live at: https://luokai25.github.io/Luo-browser/**

---

## Features

- 🖥️ Full web-based desktop UI
- 🔍 Command Palette (⌘+Space)
- 📬 Notification Center
- 🧩 Widgets (Clock, Weather, Calendar, etc.)
- 🎵 Media Player
- ⚙️ Settings Persistence
- And more!

## Keyboard Shortcuts

- `⌘+Space` - Open Command Palette
- `⌘+M` - Minimize window
- `⌘+W` - Close window

## Development

To run locally:

```bash
npm install express
node src/index.js
# Open http://localhost:3000/desktop
```
